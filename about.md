# WikiAfCReviewer

**WikiAfCReviewer** is an AI-assisted tool that helps Wikipedia editors and Articles for Creation (AfC) reviewers assess draft articles against Wikipedia's core content policies. It produces an **advisory review** — a source-by-source notability assessment (WP:GNG / SNG), neutrality (WP:NPOV) and conflict-of-interest flags, citation-quality checks, and an overall AfC-readiness summary, with the specific policy cited for each finding.

## What it does and does not do

- It **assesses** drafts and explains its reasoning. It does **not** write or rewrite article prose, and it does **not** edit, move, accept, decline, or otherwise act on any page.
- Every judgment is **advisory**. A human editor or AfC reviewer makes all decisions and performs any edits under their own account.
- It makes **no autonomous edits** to any Wikimedia project. It is read-only against the wikis.

## How it works

The tool reads the draft under review and verifies its sources. Source verification uses general web search; on-wiki reads (the draft text and related articles) use the MediaWiki Action API. The assessment is generated by a large language model (Anthropic's Claude). Because the analysis is model-generated, it can be incomplete or mistaken — it is intended to *support* a human reviewer's judgment, not replace it.

## API access and etiquette

- All wiki access is **read-only**, via the public MediaWiki Action API.
- Requests are sent serially (one at a time), rate-limited, and use the `maxlag` parameter to back off automatically when the database replicas are lagged.
- This `User-Agent` identifies the tool and links here so administrators can understand its traffic and contact the operator.
